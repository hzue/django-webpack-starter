# django-webpack-starter

* Front end preprocessor
  * jade
  * livescript
  * stylus

* Directory
  * Example  Project - `web`
  * Example App - `example`

* Set django and webpack port

  ```
  vi option.json
  ```
* Run django server

  ```
  ./runserver
  ```
* Run webpack

  * For just build file & watch
    ```
    npm run watch
    ```

  * For dev-server
    ```
    npm run dev
    ```

  * For production
    ```
    npm run prod
    ```
